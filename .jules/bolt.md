## 2024-05-24 - Prevent Carousel Hover Cascading Re-renders
**Learning:** In complex interactive UI components like carousels (`BookCarouselSection`), parent-level tracking of interaction states (e.g., `isHovered`) using `onMouseEnter` and `onMouseLeave` triggers frequent state updates. Without memoization, this forces all heavily animated child list items (like `BookCard` with framer-motion and 3D tilt effects) to needlessly re-render during every cursor movement over the boundary, wasting significant main-thread time.
**Action:** Always wrap heavy list item components (e.g. cards, list rows) in `React.memo` when rendering them inside a parent that frequently updates its local state (like for interaction tracking or scrolling), to prevent unnecessary cascading re-renders.
