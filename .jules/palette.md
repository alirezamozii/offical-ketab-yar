## 2026-06-19 - Adding ARIA labels to avatar selection buttons
**Learning:** When using Persian language in the app, the ARIA labels on elements should match the context language (e.g., 'انتخاب آواتار' for Avatar Selection). Using dynamic strings based on URLs works well for descriptive avatar labels.
**Action:** Always verify the application language context before writing ARIA strings, and ensure image-based buttons have specific alt/aria labels rather than generic ones.
